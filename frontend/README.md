# OracleAI Frontend

Professional React + TypeScript + TailwindCSS frontend with cyberpunk "Future-Fi" design.

## Setup

```bash
# Install dependencies
npm install

# Copy environment file
cp .env.example .env

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

✅ Cyberpunk design with glass morphism
✅ Real-time market updates
✅ AI chat interface
✅ Responsive design
✅ Framer Motion animations
✅ TailwindCSS styling

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Framer Motion
- Lucide Icons
- TanStack Query

Access at: http://localhost:5173
